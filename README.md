# VIT-Bhopal-Campus-Guide
#Fundamentals in AI and ML (CSA2001)

#Project Overview

The VIT Bhopal Campus Navigator is a kind of helper that assists students and visitors in finding their way around the university campus. It does this by creating a map of the campus and using this map to find the path between any two buildings, like the academic blocks, hostels and the lab complex.

This project was created to show how Search Algorithms and Intelligent Agent design can be used in life.

#AI Concepts Applied

This project uses important ideas from the CSA2001 syllabus:

Intelligent Agents: The VIT Bhopal Campus Navigator is a Goal-Based Agent that follows a simple process. It looks at the situation searches for a solution and then acts on that solution.

Search Strategies: The VIT Bhopal Campus Navigator uses a method called Breadth-First Search to find the path between two points.

Knowledge Representation: The VIT Bhopal Campus Navigator uses a graph to store information about the campus.

State-Space Search: The VIT Bhopal Campus Navigator thinks of the campus as a series of places and paths that connect them.

#Tech Stack & Requirements

The VIT Bhopal Campus Navigator knows about the following places:

The VIT Bhopal Campus Navigator knows about the academic areas, like Academic Block 1 Academic Block 2 the Lab Complex, the Library and the Research Wing.

The VIT Bhopal Campus Navigator also knows about the hostels and other places like Hostel Block A, Hostel Block B, the Canteen and the Sports Complex.
